# To-Do List CLI

โปรแกรม To-Do List สำหรับจัดการงานผ่าน Command Line Interface (CLI)

## คุณสมบัติ
- เพิ่มงานใหม่
- ดูงานทั้งหมด
- แก้ไขงาน
- ลบงาน
- ข้อมูลงานถูกบันทึกในไฟล์ tasks.json

## ขั้นตอนการติดตั้ง
1. ติดตั้ง Python (แนะนำเวอร์ชัน 3.7 ขึ้นไป)
2. ดาวน์โหลดไฟล์ main.py และไฟล์ tasks.json (จะถูกสร้างอัตโนมัติเมื่อใช้งาน)
3. เปิด Terminal หรือ Command Prompt ในโฟลเดอร์โปรเจกต์

## วิธีรันโปรแกรม
```bash
python main.py
```

## ตัวอย่างการใช้งาน
1. เมื่อรันโปรแกรม จะพบเมนูหลัก:
```
===== ToDoList Menu =====
1. เพิ่มงานใหม่
2. ดูงานทั้งหมด
3. แก้ไขงาน
4. ลบงาน
5. ออกจากโปรแกรม
เลือกเมนู (1-5):
```
2. เลือก 1 เพื่อเพิ่มงานใหม่
3. เลือก 2 เพื่อดูรายการงานทั้งหมด
4. เลือก 3 เพื่อแก้ไขงาน (เลือกหมายเลขงานที่ต้องการแก้ไข)
5. เลือก 4 เพื่อลบงาน (เลือกหมายเลขงานและยืนยันก่อนลบ)
6. เลือก 5 เพื่อออกจากโปรแกรม (ข้อมูลจะถูกบันทึกอัตโนมัติ)

## หมายเหตุ
- ข้อมูลงานจะถูกบันทึกในไฟล์ tasks.json ในโฟลเดอร์เดียวกับ main.py
- หากไม่มีไฟล์ tasks.json โปรแกรมจะเริ่มต้นด้วยรายการว่าง
